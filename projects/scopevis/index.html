ScopeVis

DOE / Brookhaven National Laboratory · Technical Lead

Overview

ScopeVis is an AI-assisted toolkit for segmenting and visualizing nanoscale microscopy data, developed in partnership with Brookhaven National Laboratory under DOE funding. The project evaluated and combined multiple computer vision approaches to automate 3D object reconstruction from 2D microscopy stack slices — a process that previously required hours of manual expert annotation per sample.

The toolkit was designed to be generalizable across microscopy modalities and sample types, with automated threshold selection and Bayesian hyperparameter optimization reducing the per-dataset configuration burden on domain scientists.

Technical Approach

  • Implemented and evaluated five 2D segmentation approaches: SAM (Segment Anything Model), MicroSAM (microscopy-adapted SAM), 3D watershed, region growing, and a custom Sobel-edge watershed pipeline
  • Graph-based clustering to join 2D segmentation masks across z-slices into coherent 3D objects, handling object split and merge cases common in dense nanoscale samples
  • Entropy-based automatic threshold selection for foreground/background separation, removing the need for per-image manual threshold tuning
  • KDE background estimation and mutual-information scoring for feature quality assessment and segmentation validation
  • TPE (Tree-structured Parzen Estimator) Bayesian optimization for pipeline hyperparameter tuning across the full segmentation and clustering configuration space

Results

  • All five technical objectives delivered at 100% completion
  • Estimated 94% time reduction compared to manual expert annotation methods
