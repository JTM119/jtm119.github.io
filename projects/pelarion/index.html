Pelarion

EPA Phase II (2023–2025) · Technical Lead

Overview

Pelarion is a full-stack systematic review platform built for the EPA to accelerate grey literature screening across regulatory documents, reports, technical studies, and web content. The platform automates the most time-consuming stages of evidence synthesis — title and abstract triage, full-text relevance assessment, and conflict flagging — while preserving reviewer control and audit trails throughout.

The EPA's systematic review workflows required handling heterogeneous source types (PDF, HTML, scanned documents), supporting multi-reviewer conflict resolution, and producing structured outputs suitable for regulatory documentation. Pelarion was designed to reduce the screening burden on domain experts without removing them from the decision loop.

Technical Approach

  • PDF and HTML ingestion pipeline with sentence-aware chunking and LLM-assisted title and abstract extraction for documents lacking structured metadata
  • BM25 lexical ranking for fast first-pass title and abstract triage across large literature sets
  • Random Forest and XGBoost classifiers with Optuna Bayesian hyperparameter tuning for learned relevance scoring at the abstract stage
  • Full-text screening via Gemini Flash LLM with structured prompts — producing criterion-by-criterion rationales and page-level citations as structured JSON, enabling reviewer audit
  • Multi-reviewer conflict detection and resolution workflow with configurable inclusion thresholds
  • Production stack: Django REST backend, React frontend, PostgreSQL, Celery task queue, RabbitMQ message broker, Redis cache, Azure Blob Storage for document storage

Results

  • 80% recall and 86% precision on held-out literature sets
  • LLM-based full-text screening with structured criterion-by-criterion outputs and page-level citations, enabling reviewer verification without re-reading full documents
  • Delivered as a production web application used by EPA reviewers across an active systematic review engagement
