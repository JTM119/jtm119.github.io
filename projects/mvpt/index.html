MVPT

NASA SBIR · Technical Contributor

Overview

MVPT (Model Validation via Precomputed Transformations) is a model-agnostic method for predicting whether an AI model's validation results will transfer to novel mission data — without rerunning the full model. Instead of re-executing expensive inference, MVPT uses fast vector lookup against a precomputed embedding database to estimate validation status, enabling rapid triage of new data against known model behavior.

The approach is particularly relevant for space mission contexts, where computational budgets are constrained, ground-in-the-loop validation cycles are slow, and distributional shift from mission data relative to training data is common.

Technical Approach

  • Proof of concept on DSCOVR L1 solar wind data using a TCN with Student-t likelihood head for probabilistic forecasting
  • Penultimate-layer embeddings extracted from validation-set inputs and indexed using HNSW (Hierarchical Navigable Small World) approximate nearest-neighbor search for sub-millisecond retrieval
  • SoftMax-weighted validation status estimation from nearest neighbors in the embedding space, producing a calibrated validation confidence score for new input windows
  • Volatile-regime grouping to stress-test validation generalization under distributional shift — evaluating whether validation status predictions hold across structurally distinct solar wind regimes

Results

  • FCCV (Fraction of Correctly Classified Validation windows) of 0.95 ± 0.05
  • Correctly predicted validation status for 19 of 20 independent volatile-regime groups
  • Completed all Phase I objectives at 83% budget utilization
