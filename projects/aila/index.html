AILA

DoD SBIR (2023–2024) · Technical Lead

Overview

AILA is a multimodal AI prototype developed for the DoD that prioritizes targeting-package review by detecting mismatches between overhead imagery and associated captions. When imagery and text diverge — through mislabeling, outdated content, or adversarial manipulation — AILA surfaces those packages for analyst attention, reducing the cognitive burden of manually triaging large review queues.

The project combined vision-language model fine-tuning, military-domain language model adaptation, and post-hoc explainability to produce a system analysts could trust and interrogate. Both saliency maps and LIME word-importance outputs are delivered alongside each prediction, showing which image regions and caption words drove the mismatch score.

Technical Approach

  • Fine-tuned BLIP (Bootstrapping Language-Image Pre-training) on a 1,435-pair dataset of overhead and satellite imagery spanning four target categories
  • Developed milBERT via the ExBERT architecture to adapt BERT to military text through domain-specific masked language modeling, improving top-1 accuracy from 0.394 to 0.498 on military corpora
  • Combined vision and language scores to produce a per-package mismatch probability, with configurable thresholds for analyst review queues
  • Saliency maps generated for image regions and LIME word-importance charts generated for caption tokens, delivered as per-result explanation outputs
  • React frontend, Django and Flask backends, PostgreSQL, Kubernetes orchestration, NGINX ingress, GPU-backed model serving

Results

  • 81.9% accuracy, 87.2% recall, 79.2% precision on held-out evaluation sets
  • Simulated 4x+ reviewer speedup over manual review baseline under the evaluated queue prioritization scheme
  • milBERT top-1 masked language modeling accuracy: 0.394 → 0.498 on military text
  • Explainability outputs (saliency + LIME) delivered alongside every prediction
