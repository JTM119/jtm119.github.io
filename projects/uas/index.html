UAS Certification

NASA Phase I SBIR (2022–2023) · Technical Contributor

Overview

This NASA Phase I SBIR developed a Verification and Validation (V&V) Testing Environment (TE) for certifying autonomous UAS path-planning algorithms, conducted in partnership with the University of Cincinnati. The TE enables systematic injection of sensor and perception failures to evaluate UAS responses under conditions that would be unsafe, impractical, or prohibitively expensive to test in real flight.

The certification challenge for autonomous UAS is that traditional flight testing cannot feasibly cover the tail of failure scenarios that regulators require evidence for. Simulation-based TE approaches allow exhaustive failure injection at scale, but must be validated as faithful proxies for real-world behavior. This project developed and validated that proxy relationship.

Technical Approach

  • Microsoft AirSim simulation environment built on Unreal Engine 4, providing high-fidelity flight physics and sensor simulation for multiple UAS platforms and path-planning algorithms
  • GPS corruption injection to test path-planning resilience under navigation failure — simulating satellite outage, spoofing, and degraded signal conditions
  • ML perception input corruption via Gaussian blur and pixel obscuration to simulate camera degradation from environmental conditions or sensor damage
  • Monte Carlo trial methodology: 80 simulated flights per path-planning algorithm under each failure condition, providing statistically robust failure detection rates
  • Cross-validation of TE failure detection rates against independent ML model predictions across five camera corruption conditions, establishing the TE as a reliable proxy for real-world UAS response

Results

  • GPS failure detection demonstrated across all path-planning algorithms under test, with statistically robust rates from 80-flight Monte Carlo trials per algorithm
  • TE failure detection rates matched independent ML model predictions across all five camera corruption conditions, validating the TE as a reliable proxy for real-world performance
