GRIDview

DOE Phase I (2025–2026) · Technical Lead

Overview

GRIDview is an uncertainty-aware decision support prototype for electric grid operations, developed under DOE Phase I SBIR. The platform enables grid operators to make simulation-based decisions informed by calibrated probabilistic forecasts for load, solar, and wind — surfacing not just expected values but full predictive distributions with quantified uncertainty across short planning horizons.

The core research question was whether calibrated probabilistic forecasts could be integrated into an operable decision-support interface without sacrificing latency or usability. The result is a system that generates correlated multi-variable scenario draws, renders them as interactive fan charts, and delivers them end-to-end in under one minute.

Technical Approach

  • Benchmarked Gaussian Processes, Bayesian Neural Networks, and Temporal Convolutional Networks (TCN) with Student-t likelihood heads for probabilistic load, solar, and wind forecasting
  • Selected TCN with Student-t head as primary model based on calibration-accuracy tradeoff across held-out evaluation sets
  • Gaussian copula scenario generation used to draw correlated samples across load, solar, and wind simultaneously — preserving cross-variable dependencies that independent marginal draws would miss
  • Evaluated interval coverage at 90%, 95%, and 99% nominal levels using CRPS as the primary scoring rule
  • FastAPI backend serving model inference; PyTorch for training and serving; React and D3.js for interactive fan chart and confidence band visualization
  • Containerized with Docker, deployed via Azure DevOps CI/CD pipeline

Results

  • 27.1% CRPS improvement over deterministic baseline
  • Calibrated coverage at 90%, 95%, and 99% prediction intervals
  • End-to-end latency under one minute from data ingestion to rendered visualization
