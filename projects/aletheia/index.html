Aletheia

U.S. Air Force — Technical Contributor

Overview

Aletheia is an image forensics system for detecting raster manipulation and AI-generated synthetic imagery. The project addressed two related but distinct problems: detecting post-capture edits (splicing, copy-move, inpainting) where a real image has been locally altered, and identifying fully AI-generated images where no original photograph exists.

Both capabilities are relevant for intelligence analysis and content verification contexts where the authenticity of visual evidence cannot be assumed. The system delivers pixel-level manipulation localization alongside document-level binary classification.

Technical Approach

  • CAT-Net (Compression Artifact Tracing Network) for pixel-level localization of manipulation artifacts — detecting JPEG compression inconsistencies introduced by post-capture editing tools
  • Deep consensus aggregation for combining evidence across multiple detection pathways into a single manipulation probability estimate
  • Secondary classification layer for manipulation type identification: splicing (inserting content from another image), copy-move (duplicating regions within the same image), and inpainting (filling removed regions with synthesized content)
  • CLIP-based experiments for AI-generated image detection, leveraging vision-language alignment representations to distinguish real photographs from synthetic imagery
  • Evaluation using pixel-level precision, pixel-level recall, and Cohen's kappa to assess both detection and localization quality

Results

  • 98.3% manipulation detection precision, 97.4% recall across splicing, copy-move, and inpainting categories
  • 98.7% binary real/fake accuracy on a 12,000-image AI-generated image test set using CLIP-based detection
