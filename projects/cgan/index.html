CGAN / Eikon

DoD / AFRL · Technical Lead

Overview

CGAN / Eikon is an AFRL-funded synthetic satellite imagery generation effort aimed at augmenting labeled training data for object detection. The program addressed a core constraint in overhead imagery AI: labeled satellite data is expensive, access-controlled, and sparse — making it difficult to train detectors that generalize across geography, sensor, and seasonal conditions.

The project developed controllable generation pipelines for realistic synthetic overhead imagery and systematically evaluated whether training on synthetic data improves or degrades detector performance across a large grid of model configurations.

Technical Approach

  • Conditional GAN architecture for controlled synthetic imagery generation from semantic feature maps, enabling targeted generation of specific terrain types and object configurations
  • map2sat translation pipeline (semantic map → satellite imagery) as the primary generation approach, producing photorealistic overhead imagery from vector geospatial inputs
  • Feature generation modules for roads, water bodies, biomes, and building footprints, supporting controllable scene composition
  • Systematic augmentation experiments across 300 Faster R-CNN model variants and 220 YOLO model variants, each trained with varying ratios of real versus synthetic data to characterize augmentation effects
  • Reusable testing infrastructure for ongoing evaluation of synthetic-versus-real training tradeoffs across different detector architectures and data regimes

Results

  • Delivered 300 Faster R-CNN and 220 YOLO model variants with characterization of synthetic augmentation effects across data ratios
  • Identified conditions under which synthetic augmentation improves versus degrades detector performance, informing data collection strategy for follow-on programs
  • Reusable evaluation infrastructure delivered for continued program use
